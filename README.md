# Guia para usar la aplicación

## TECNOLOGIAS USADAS:
* Typescript
* Angular
* Node.js
* PostgreSQL

* De primero tienes que clonar el repositorio y abrir una terminal con el proyecto ya abierto.
## Para Configurar Backend:
* Tienes que tenes PostGreSQL instalado y escribir "CREATE DATABASE appprclientes;"
* Tienes que editar el archivo .env y adaptarlo a tu maquina
*  Luego de tener abierto el proeyecto tienes que escribir en la terminal "pnpm install" en la carpeta de Backend
*  Y luego escribir "pnpm dev" y mostrará un mensaje diciendo "Servidor Backend escuchando en el puerto 3000"
##   Para Configurar Frontend:
* Tienes que escribir en ** otra terminal ** y en la carpeta ubicada en el Frontend "pnpm install"
* Y tambien "pnpm exec ng serve"

## Para terminar
Tienes que escribir en el navegador "http://localhost:4200" y con eso tendras la aplicacion corriendo correctamente
